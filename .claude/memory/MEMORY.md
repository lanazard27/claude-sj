# 에이전트 운영 교훈
- **병렬 스폰 3개+면 code-writer가 파일 누락** — 항상 파일 존재 확인 후 재스폰
- **비표준 라이브러리는 API 사용법을 프롬프트에 포함** — 타입 선언을 에이전트가 임의로 만들면 잘못된 시그니처 → 런타임 버그
- **에이전트당 3-6파일이 품질 적정, 7개+ 급락**
- **스펙 단위 수직 분할 > 레이어별 수평 분할**
